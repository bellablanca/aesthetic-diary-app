# Aesthetic Diary App 💖
A cute, minimalist desktop diary built with Python and Tkinter.

## Features
- Pink-themed UI 🌸
- Save diary entries as text files
- Upload and store pictures
- Menu bar with font selector and future expandability
- Packaged as a Windows .exe
